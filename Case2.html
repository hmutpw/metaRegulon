Decoding the dynamic TF activity during HSC formation

1. Introduction

In this case study, we systemically estimated the dynamic TF activity during HSC formation. We use the single-cell RNA-seq data download from Hou et.al. 2020

1.1 loading data

The single-cell RNA-seq data of endothelial cells (ECs) were firstly downloaded from GEO, and then we only choose three cell populations for further analysis (including early-AECs, late-AECs, and HECs). The TF activity analysis were performed using the standard metaRegulon pipelines except that the 06_endothelial_cells.txt.gz were used as prior-network. Moreover, the pseudotime information of each single-cell were obtained directly from original paper. Here we start with the analysis steps from TF activity matrix, which can be download from Zenodo.

Firstly, we need to load the SingleCellExperiment object that generated from metaRegulon() function.

suppressMessages(suppressWarnings(library(scATFR)))
suppressMessages(suppressWarnings(library(metaRegulon)))
atfr <- readRDS("D:/HSC_Autophagy/Markdown/5.metaRegulon/metaRegulon/inst/extdata/EHT/atfr.rds")

1.2 Confirming the cell paths from pseudotime analysis output

One of the most important steps of dynamic TF activity analysis is to identify cell paths from your single-cell RNA-seq data after pseudotime analysis (e.g Monocle). In this study, we just get this information from original paper. Therefore, we first take a look at the pseudotime path from the paper.

cell_anno <- as.data.frame(colData(atfr))

We can see that there are two paths in above result (earlyAEC to HEC and earlyAEC to lateAEC). However, we found that the cell trajectory is a continuous state and some cells may be in transition state. Next, we identified the cell at the branch point in above two paths and generated the pseudotime weight matrix:

pseudotime_list <- readRDS("D:/HSC_Autophagy/Markdown/5.metaRegulon/metaRegulon/inst/extdata/EHT/cell_weight_pseudotime.rds")
cell_weight <- pseudotime_list$weight
cell_pseudotime <- pseudotime_list$pseudotime

The cell_weight indicate the cells belong to which path and the cell_pseudotime have the pseudotime value of each cell in corresponding path.

In this section, we perform differential analysis on TFs at activity level

We using unsupervised hierarchical clustering method to cluster these differential TFs and seperated them into 6 patterns. Each pattern represent a group of TFs that share similar dynamic activities. Moreover, these regulons may have divergent function between HECs and lateAECs. Next, we estimate the average activity of TFs in each group and visualize them with curves.

2.2 Correlations between TF activity and TF expression

In this section, we evaluated the correlations between TF activity and TF expression in different patterns.

# create the expression level object
exp_mat <- assay(atfr,"logcounts")[row.names(diff_gene_res),]
exp_sce <- fitGAM(counts = exp_mat, pseudotime = cell_pseudotime, cellWeights = cell_weight)
# smooth gene expression
ysmooth_expression <- predictSmooth(models = exp_sce, gene = row.names(exp_sce), nPoints = 100)
# get two lineage dynamic expression level
ysmooth_expression_lineage_1 <- ysmooth_expression %>% filter(lineage==1)
ysmooth_expression_lineage_1_tbl <- reshape2::dcast(data = ysmooth_expression_lineage_1[,-1], formula = gene~time)
## Using yhat as value column: use value.var to override.
ysmooth_expression_lineage_1_mat <- as.matrix(ysmooth_expression_lineage_1_tbl[,-1])
row.names(ysmooth_expression_lineage_1_mat) <- ysmooth_expression_lineage_1_tbl$gene

ysmooth_expression_lineage_2 <- ysmooth_expression %>% filter(lineage==2)
ysmooth_expression_lineage_2_tbl <- reshape2::dcast(data = ysmooth_expression_lineage_2[,-1], formula = gene~time)
## Using yhat as value column: use value.var to override.
ysmooth_expression_lineage_2_mat <- as.matrix(ysmooth_expression_lineage_2_tbl[,-1])
row.names(ysmooth_expression_lineage_2_mat) <- ysmooth_expression_lineage_2_tbl$gene

ysmooth_expression_lineage_1_scale <- t(apply(ysmooth_expression_lineage_1_mat ,1,function(x){x-x[1]}))
ysmooth_expression_lineage_2_scale <- t(apply(ysmooth_expression_lineage_2_mat,1,function(x){x-x[1]}))
diff_gene_exp_matrix <- cbind(ysmooth_expression_lineage_1_scale[row.names(diff_gene_res), rev(colnames(ysmooth_expression_lineage_1_scale))], ysmooth_expression_lineage_2_scale[row.names(diff_gene_res),])

# calculate the collection coefficient between activity level and expression level.
cor_val <- c()
for(i in row.names(diff_gene_res)){
  cor_i <- cor(diff_gene_matrix[i,],diff_gene_exp_matrix[i,])
  cor_val[i] <- cor_i
}

# merge TFs with cluster and correlation information
diff_gene_cor_tbl <- data.frame(anno_row,cor=cor_val[row.names(anno_row)])
diff_gene_cor_tbl$cluster <- factor(diff_gene_cor_tbl$cluster,levels = rev(levels(diff_gene_cor_tbl$cluster)))
library(ggridges)
d <- ggplot(data = diff_gene_cor_tbl, mapping = aes(x=cor, y=cluster, fill=cluster))+
  geom_density_ridges_gradient(size = 1, scale = 1.4, panel_scaling = F, alpha=.8,na.rm=T)+
  scale_x_continuous(limits = c(-1,1))+
  theme_bw()
d
## Picking joint bandwidth of 0.16

We found that most differential TFs showed relatively high correlation between expression and activity level. However, TFs in some clusters showed relatively low correlations, indicating that TFs in these two clusters may showed some controversial functions. For example, the target genes of these TFs may be repressed. Collectively, activity level analysis is more likely to identify more TFs that potentially regulate the cell fate decision.
